# Breaking Bad Algebra

A React + Vite app for guided algebra practice.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a GitHub repo and upload these files.
2. Go to Vercel and import the GitHub repo.
3. Framework preset: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`
6. Deploy.

## Notes

This starter version supports:
- simple linear equations
- two-step equations
- simple bracket equations

Examples:
- `x + 5 = 12`
- `3x + 5 = 20`
- `2x - 6 = 10`
- `2(x + 3) = 14`
